Created detection rules for brute-force and beaconing behavior mapped to **MITRE ATT&CK**.  

**Key Achievements:**  
- Correlated 500K+ logs in real time  
- Developed dashboards for login and network monitoring

**Tools:** Splunk, Sysmon, Zeek, Python SDK
